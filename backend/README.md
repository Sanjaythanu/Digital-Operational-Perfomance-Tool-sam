# DIGITAL OPERATIONAL PERFORMANCE TOOL (DOPT)

## Project Structure
- `frontend/`: React Application
- `backend/`: Node.js/Express Application

## How to Run

### Backend
1. `cd backend`
2. `npm install`
3. Create `.env` file based on `.env.example`
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

## Roles
- **Admin**: Full access to all data and user management.
- **User**: Access only to own operations and analytics.
